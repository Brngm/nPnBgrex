

TO INSTALL THE LIBRARY VascoAsturiano-libs (already done # no need to do it again)
see https://github.com/vasturiano/ & https://github.com/vasturiano/react-force-graph/tree/master

🔷1 Download the file: force-graph.min.js
    1 Go to 👉 https://cdn.jsdelivr.net/npm/force-graph/dist/force-graph.min.js
    2 Click on the file
    3 Right-click → Save As...
    4 Save it in the folder: VascoAsturiano-libs/2d-force-graph.js

🔷2 Download the file: 3d-force-graph.js
    1 Go to 👉 https://cdn.jsdelivr.net/npm/3d-force-graph/dist/3d-force-graph.js
    2 Click on the file
    3 Right-click → Save As...
    4 Save it in the folder: VascoAsturiano-libs/3d-force-graph.js

🧠 Advantages of the method:
    ✅ No network requests. Everything is in the .html file.
    ✅ Supports large files. The browser reads the text without blocking.
    ✅ Clear code. Data is separated from JS code.
    ✅ Compatible with python3 -m http.server.
